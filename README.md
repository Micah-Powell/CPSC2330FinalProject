# CPSC2330 Final Project

This web application displays a list of members of the **Anderson University Pickleball Club**.

## Features
- **Add Members:** Enter a member's name and email in the input fields, then click **Submit** to add them to the list.
- **Remove Members:** Click the **X** button on the same row as a member to remove them from the list.

## Installation Instructions

1. Install the **Flask** library for Python:

2. Download all files in this repository **except the README**.

3. Ensure all files are placed on your computer with the same directory structure as in the repository.

4. Run the server by executing:
   python Server.py

5. While the server is running, open a web browser and navigate to:
   http://127.0.0.1:5000

The site should now be running and functioning as intended.
